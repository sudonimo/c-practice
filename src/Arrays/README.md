# Arrays
## Arrays are fixed length containers.

- array.cc shows the basic operations with Arrays (initialization, traversing, and element accessing).
- sum-array.cc It is the accumulated sum of one Array.
- find-primes finds the prime numbers that the Array contains.
- find-item.cc shows a template function that finds a value into the array. Template functions operate  with any type of input and return values.
- close20.cc. finds the closest to 0 value, It may be negative or positive.
- increment.cc shows the difference between prefix increment and the postfix increment by operator overloading.
  also shows the python like printing.
